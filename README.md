# Flutter Real Time Chat app with Socket.io & MongoDB

Service : https://github.com/i-Senku/Chat-Socket-Service

## Features

- Online/Ofline User Status
- Auto Scroll ListView
- Check Room Control
- Show Image
- Send Image
